# pjsip-opus-ssl

## Description
PJSIP 2.6 with Opus 1.2.1 Codec Support and OpenSSL 1.0.2l


## Building

Just run ./build.sh

## Installation

pjsip-opus-ssl is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod 'pjsip-opus-ssl'
